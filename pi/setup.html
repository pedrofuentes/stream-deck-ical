Features
  • New: Support for recurring events (daily, weekly, monthly, etc.)
  • New: Configurable time window (1, 3, 5, or 7 days)
  • New: Improved timezone handling for Outlook, Google, and Apple calendars
  • Checks for updates every 10 minutes
  • When a new URL is saved, an immediate update is triggered
  • "Force Refresh" can be used to manually trigger an update

Time Left

  • Shows time left until the current meeting ends
  • Changes icon color to orange when 5 minutes are left
  • Changes icon color to red when 30 seconds are left (stays red for 5 minutes after meeting ends)
  • Supports multiple concurrent meetings - push the button to cycle through them
  • Shows meeting indicator (e.g., "1/3") when multiple meetings are active

Next Meeting

  • Shows countdown to next meeting start time
  • Push the button to display the meeting title (scrolling marquee)
  • Push again to return to countdown display
  • Changes icon color to orange with 5 minutes remaining
  • Changes icon color to red with 30 seconds remaining
Calendar Compatibility

This plugin supports iCal feeds from:

  • Google Calendar: Export your calendar and use the secret iCal URL
  • Microsoft Outlook / Office 365: Automatic timezone conversion for Windows timezones
  • Apple Calendar: Full support for iCloud calendar feeds
  • Any other calendar application that provides an iCal (.ics) feed
